# 🔢 Conversor de Bases Numéricas

¡Hola! Este es un proyecto que estoy construyendo como estudiante de Ciencias de la Computación para aplicar conceptos de sistemas numéricos y desarrollo web.

## 🚀 Objetivos del proyecto
- Convertir números entre bases: Decimal, Binario, Octal y Hexadecimal.
- Crear una interfaz web limpia y amigable.
- Practicar la manipulación del DOM con JavaScript puro.

## 🛠️ Tecnologías
- HTML5
- CSS3 (Próximamente)
- JavaScript (Vanilla)
